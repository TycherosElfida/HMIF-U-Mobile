HMIF U-Mobile
Aplikasi Manajemen Himpunan Mahasiswa Informatika

=== DAFTAR AKUN (ROLE) ===
Berikut adalah daftar role akses yang tersedia dalam aplikasi:

1. President (Ketua Himpunan)
   - Akses: Full Access (Admin Panel, Finance, Secretariat).
   - Fitur Khusus: 
     * Manajemen Role User (Menu "Manage Users" di Admin Panel).
     * Approval Proposal & LPJ di Dashboard Sekretariat.
     * Monitoring Keuangan.

2. Secretary (Sekretaris)
   - Akses: Admin Panel -> Secretariat Dashboard.
   - Fitur Khusus:
     * Mengajukan (Submit) Proposal dan Laporan Pertanggungjawaban (LPJ) berupa Link/URL.
     * Memantau status approval dokumen (Pending/Approved/Rejected).

3. Treasurer (Bendahara)
   - Akses: Admin Panel -> Finance Dashboard.
   - Fitur Khusus:
     * Mencatat Pemasukan dan Pengeluaran Kas.
     * Melihat rekapitulasi dan riwayat transaksi keuangan.

4. Member (Anggota)
   - Akses: Halaman Utama (Home), Events, Profile.
   - Fitur Khusus:
     * Melihat timeline Pengumuman (Announcements) terbaru.
     * Mendaftar ke Event yang tersedia.
     * Mengakses Digital ID Card di profil.

=== FITUR UTAMA APLIKASI ===
1. Manajemen Event & Pengumuman
   - Admin dapat membuat, mengedit, dan mempublikasikan info kegiatan.
   - Anggota dapat melihat detail dan mendaftar event.

2. Digital Secretariat (Sekretariat)
   - Sistem pengajuan surat/dokumen (Proposal/LPJ) berbasis URL (Google Drive/Docs).
   - Workflow persetujuan (Approval) oleh Ketua Himpunan secara real-time.

3. Transparansi Keuangan (Finance)
   - Pencatatan arus kas masuk dan keluar yang dikelola oleh Bendahara.
   - Dashboard monitoring untuk Ketua Himpunan.

4. Manajemen Pengguna (Role Management)
   - Ketua Himpunan dapat mengatur hak akses anggota (menjadikan staff, bendahara, atau sekretaris) langsung dari aplikasi.
